What is AirNote?

AirNote is the only not-WebView-based WYSIWIG HTML Editor
for Android and iOS.

WYSIWYG HTML editor

  • Full HTML 4.1 and CSS 2.1 implemented
  • Minimum HTML 5 and CSS 3.0 features for layout compatibility.

100% pure Java/Objective-C component

  • No WebKit used
  • No inter-process communication
  • No Javascript

Easy UI customizing & integration

  • Synchronous DOM manipulation
  • Easy event handling

AirNote provides full WYSIWYG HTML editing features as powerful as the PC version of Gmail.
AirNote provides more precise HTML+CSS rendering quality than Microsoft Word.
